# 新建对话技能

用户想要在故事「{book_name}」中开始一个新对话。

## 流程

### 第一步：询问是否复用开场白

先告诉用户这个故事的已有开场白，然后问是否复用：

**"你想开始一个新对话。这个故事的已有开场白：**

**1. {opening_1_name}**
**2. {opening_2_name}**
**...**

**你想要复用其中一个开场白，还是重新创建一个？**

- **复用**：回复编号或名字，我直接用那个开场白开始新对话
- **重建**：回复「新建」或「重新创建」，我们一步步来设定"

### 第二步A：复用

用户选择复用 → 调用 `reuse_opening` 函数，传入对应的 `dialogue_id`。

函数执行成功后，告诉用户："新对话已创建！关闭这个窗口，在对话界面中继续吧。"

### 第二步B：重建

用户选择重建 → 切换到完整的开场白创建流程（参考 open-dialogue 技能）：

逐步询问：时间 → 地点 → 主角 → NPC → 人称 → 模式

收集完毕后调用 `start_dialogue` 创建新对话。
